HTML é uma linguagem de marcação usada para criar conteúdo na Web. Define a estrutura e o conteúdo de um documento. Usa elementos, tags e atributos para definir a aparência e comportamento.
CSS é uma linguagem de estilo usada para definir a aparência e o layout de páginas da Web. É usada para estilizar HTML e outros tipos de documentos. O CSS permite definir cores, fontes, margens, alinhamentos e outras propriedades visuais.
JavaScript é uma linguagem de programação para conteúdo interativo na Web. Usada com HTML e CSS, adiciona funcionalidade e interatividade. Permite manipular elementos, lidar com eventos e criar animações e jogos.
jQuery é uma biblioteca JavaScript que simplifica a manipulação de elementos HTML e CSS. Permite escrever código mais eficiente e reduzir a quantidade de código. É amplamente utilizada em desenvolvimento web para interatividade, animações e manipulação de formulários.
Bootstrap é um framework front-end de código aberto. Possui biblioteca de componentes pré-construídos para desenvolvimento web responsivo. Ampla utilização na indústria de desenvolvimento web.
Git é um sistema de controle de versão distribuído. Permite rastrear alterações em arquivos e colaboração entre membros da equipe. Amplamente utilizado na indústria de desenvolvimento de software para facilitar o trabalho em equipe, organização de código e integração contínua.
